# Simulador de Financiamento

Aplicação web desenvolvida para simular financiamento de veículos com cálculo automático de parcelas, juros e valor total pago.
## Funcionalidades

✔ Cálculo automático de financiamento  
✔ Sistema de amortização com juros compostos  
✔ Cálculo de:
- Valor financiado
- Valor da parcela
- Total pago
- Total de juros

✔ Validação de campos  
✔ Formatação automática em moeda (R$)  
✔ Layout moderno e responsivo  
✔ Interface intuitiva
